Sistema de control de inventarios desarrollado para el negocio familiar de venta de fruta de temporada.

Permite registrar entradas, salidas y ventas en tiempo real, con resúmenes automáticos al final de cada jornada. La interfaz está en Excel para facilitar el uso por parte del equipo; la lógica de procesamiento y la base de datos corren en Python y SQLite.
